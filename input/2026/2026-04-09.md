## トピックス

### MCP C# SDK v1.0.0 リリース

MCP (Model Context Protocol) C# SDK の初のステーブルリリースとなる v1.0.0 がリリースされた。
また、公式ブログでも [Release v1.0 of the official MCP C# SDK](https://devblogs.microsoft.com/dotnet/release-v10-of-the-official-mcp-csharp-sdk/) として紹介されている。

### .NET 11 Preview 2

[.NET 11 Preview 2 がリリース](https://devblogs.microsoft.com/dotnet/dotnet-11-preview-2/)された。Preview 3 では C# 15 の早期プレビューとして union 型 (`union Pet(Cat, Dog, Bird) { ... }`) がマージされるなど、新機能の開発が進んでいる。

### .NET March 2026 サービシングアップデート

[.NET および .NET Framework の 2026年3月のサービシングアップデート](https://devblogs.microsoft.com/dotnet/dotnet-and-dotnet-framework-march-2026-servicing-updates/)がリリースされた。また macOS デバッガー修正の [.NET 10.0.5 アウトオブバンドリリース](https://devblogs.microsoft.com/dotnet/dotnet-10-0-5-oob-release-macos-debugger-fix/)も提供されている。

### ReSharper for Visual Studio Code リリース

[ReSharper for Visual Studio Code, Cursor, および互換エディター](https://blog.jetbrains.com/dotnet/2026/03/05/resharper-for-visual-studio-code-cursor-and-compatible-editors-is-out/)がリリースされた。また [Rider 2026.1 RC](https://blog.jetbrains.com/dotnet/2026/03/20/rider-2026-1-release-candidate/) および [ReSharper 2026.1 RC](https://blog.jetbrains.com/dotnet/2026/03/20/resharper-2026-1-release-candidate/) もリリースされ、C# 15 の早期サポートやパフォーマンス向上などが含まれている。

### MAUI Avalonia Preview 1

[MAUI Avalonia Preview 1](https://avaloniaui.net/blog/maui-avalonia-preview-1) がリリースされた。また Avalonia の商用コントロールであった WebView が [Avalonia 12 の一部としてオープンソース化](https://avaloniaui.net/blog/the-avalonia-webview-is-going-open-source)されることも発表された。

### PowerShell 7.6 (LTS) GA リリース

[PowerShell 7.6 (LTS) が一般利用可能 (GA)](https://devblogs.microsoft.com/powershell/announcing-powershell-7-6/) となった。.NET 10 (LTS) ベースとなり、タブ補完の向上やネイティブコマンドハンドリングの改善などが含まれている。

### Microsoft.Data.SqlClient 7.0 リリース

[Microsoft.Data.SqlClient 7.0](https://techcommunity.microsoft.com/blog/sqlserver/microsoft-data-sqlclient-7-0-is-here-a-leaner-more-modular-driver-for-sql-server/4503173) がリリースされた。Azure/Entra 認証コードのモジュール化による軽量化、パケット多重化による非同期読み取り性能向上 (プレビュー)、.NET 10 サポートなどが含まれている。

### dotnet/runtime での Copilot Coding Agent 10ヶ月の振り返り

[dotnet/runtime リポジトリで GitHub Copilot Coding Agent を10か月使用した結果の振り返り](https://devblogs.microsoft.com/dotnet/ten-months-with-cca-in-dotnet-runtime/)が公開された。PR の作成・マージ数や割合、こなしやすいタスクと難しいタスクの傾向などが紹介されている。

### TechEmpower Framework Benchmarks の終了

様々な言語・フレームワークのベンチマークを公開していた [TechEmpower Framework Benchmarks の終了](https://github.com/TechEmpower/FrameworkBenchmarks/issues/10932)がアナウンスされた。
